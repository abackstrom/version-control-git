Using git

Basics

git initNew repository
git cloneClone existing repo
git add fileAdd files to checkin
git commitCommit added files
git show objectNew repository
git statusShow modified files
git diffDiff local changes

Branching and Merging

git branchList branches
git checkout nameSwitch to branch
git merge nameMerge branch changes
git pull origin masterFetch remote and merge
git push origin masterPush new local commits
